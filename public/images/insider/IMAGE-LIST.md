## 📋 HƯỚNG DẪN SỬ DỤNG

1. Tải ảnh từ Google Images hoặc chụp ảnh của bạn
2. Đặt tên file đúng theo danh sách trên
3. Lưu vào thư mục `/public/images/insider/`
4. Cập nhật code trong `InsiderTipsThankYou.jsx`:
   - Tìm dòng có `image: null // TODO: Add your photo: [tên-file]`
   - Đổi thành: `image: "/images/insider/[tên-file]"`

Ví dụ:

```javascript
// Trước:
image: null // TODO: Add your photo: food-1-com-tam.jpg

// Sau khi thêm ảnh:
image: "/images/insider/food-1-com-tam.jpg"
```
